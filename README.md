# ddu-source-tab

tab source for ddu.vim

go to selected tab

# Required

## denops.vim

https://github.com/vim-denops/denops.vim

## ddu.vim

https://github.com/Shougo/ddu.vim

# Example

```
call ddu#start(#{ sources: [#{ name: 'tab' }] })
```
